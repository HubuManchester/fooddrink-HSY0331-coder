# Food Journal - Food & Drink App

## First Submission: Main Interface (List View)

### Implemented Features
- Food list display using CollectionView
- Food card design with nutrition information
- Pull-to-refresh functionality
- Add new food record
- View food details
- Dark/Light theme support
- Large text accessibility support

### Technology Stack
- .NET MAUI 8.0
- C# / XAML
- Local cache + Mock API support

### How to Run
1. Open the solution in Visual Studio 2022
2. Select an Android emulator or physical device
3. Press F5 to run the application

### Project Structure
- `Models/` - Data models (FoodItem)
- `Services/` - Data services (API + local cache)
- `Pages/` - XAML pages and code-behind files
- `Resources/Styles/` - Global style definitions

### First Submission Content
This submission focuses on the main interface list display. Future versions will include:
